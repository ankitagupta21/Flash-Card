Application contains three folders-
application-
for controllers,models and database
static
contains app.js
templates
contains all the html templates

app.db is the amin database

First user has to sign in or log in.
After that user has option to add deck or view decks.
In view deck, there is deck table with score.
User can edit deck, update deck or edit card.
User can export CSV file of decks.
In view cards, user can view all the cards.
